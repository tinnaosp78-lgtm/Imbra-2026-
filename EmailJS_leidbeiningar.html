EMAILJS LEIÐBEININGAR

EmailJS gerir þér kleift að senda tölvupóst beint úr HTML skránni án bakenda. Nemandinn fær sjálfkrafa staðfestingarpóst um leið og hann smellir "Skrá sig". Uppsetningin tekur um 5–10 mínútur og er ókeypis upp að 200 póstur á mánuði.
1
STOFNAÐU REIKNING Á EMAILJS.COM

Farðu á emailjs.com og smelltu á "Sign Up Free".

Skráðu þig með Gmail eða Outlook reikningnum þínum.

2
TENGDU TÖLVUPÓSTÞJÓNUSTU (EMAIL SERVICE)

Í valmyndinni vinstra megin smelltu á "Email Services" → "Add New Service".

Veldu Gmail eða Outlook — hvort sem þú notar.

Gefðu þjónustunni nafn, t.d. IMBRA2026, og smelltu "Connect Account" til að veita EmailJS aðgang.

📋 Afritaðu Service ID — þú þarft hann seinna. Lítur svona út: service_abc123
3
BÚÐU TIL SNIÐMÁT (EMAIL TEMPLATE)

Smelltu á "Email Templates" → "Create New Template".

Fylltu inn svona:

Subject (efnislína):

IMBRA 2026 — Skráning staðfest, {{to_name}}!

To Email:

{{to_email}}

Content (meginmál) — veldu HTML editor í EmailJS og límdu inn:

5
SETTU LYKLANA INN Í SKRÁNINGARKERFIÐ

Opnaðu imbra2026_skraning.html í textaritli (t.d. Notepad, VS Code).

Finndu þessar þrjár línur efst í JavaScript hlutanum og skiptu um YOUR_... gildin:

const EMAILJS_PUBLIC_KEY = 'aBcDeFgHiJkLmNoP'; // þitt Public Key const EMAILJS_SERVICE_ID = 'service_abc123'; // þitt Service ID const EMAILJS_TEMPLATE_ID = 'template_xyz789'; // þitt Template ID

Vistaðu skrána og hladdu upp á netið.

6
PRÓFAÐU KERFIÐ

Opnaðu skráningarkerfið, skráðu þig með þínu eigin netfangi og smelltu "Staðfesta skráningu".

Þú ættir að fá póst á nokkrum sekúndum. Ef póstinn kemur ekki — athugaðu að lyklarnir séu rétt afritaðir (engir auðir bilstafir).

💡 Ábending: EmailJS gefur þér 200 ókeypis póstur á mánuði. Fyrir 200–300 nemendur þarftu annaðhvort að uppfæra í greiddan reikning (ca. $15/mán) eða nota tvo reikninga.

✅ ALLT TILBÚIÐ!

Þegar lyklarnir eru settir inn mun kerfið senda sjálfkrafa staðfestingarpóst til hvers nemanda um leið og hann skráir sig.
