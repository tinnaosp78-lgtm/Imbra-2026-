1
HLAÐAÐU UPP ÖLLUM SKRÁNUM

Þú ert þegar inni á repository síðunni þinni Imbra-2026-.

Smelltu á Add file ▾ → veldu "Upload files"

Dragðu allar HTML skrárnar inn í gluggann eða smelltu "choose your files":

📄 imbra2026_dagskra.html
📄 imbra2026_skraning.html
📄 imbra2026_postur_forskodum.html
📄 EscapeRoom_Kennari.html
📄 EscapeRoom_Nemendur.html
📄 EmailJS_leidbeiningar.html
📄 GoogleForms_leidbeiningar.html

Skrunaðu niður og smelltu á græna hnappinn Commit changes

2
KVEIKTU Á GITHUB PAGES

Smelltu á ⚙ Settings flipann efst til hægri á síðunni.

Í vinstri valmyndinni — smelltu á "Pages" (þarf að skruna niður aðeins).

Undir "Branch" — smelltu á fellivalmyndina sem segir "None" og veldu "main".

Smelltu á Save

⏱ Bíddu 1–2 mínútur — GitHub er að setja upp síðuna þína. Endurhlöðu síðuna og þú munt sjá grænan reit með hlekknum!

4
UPPFÆRA SKRÁR SÍÐAR

Þegar þú vilt uppfæra skrá (t.d. bæta við Google Forms hlekk):

Farðu á repository síðuna → smelltu á skrána → smelltu á blýantstáknið ✏️ til að breyta — eða smelltu á Add file ▾ → "Upload files" og hladdu upp nýju útgáfunni.

💡 Ef þú hleður upp skrá með sama nafni mun hún skrifa yfir gamla útgáfuna sjálfkrafa.
5
DEILDU MEÐ NEMENDUM

Þú getur deilt hlekkjunum á Innu, í tölvupósti eða prentað sem QR kóða.

Til að búa til QR kóða — farðu á qr-code-generator.com, límdu inn hlekk og sæktu mynd af QR kóðanum.

💡 Nemendur þurfa ekkert að skrá sig inn — hlekkurinn er opinn fyrir alla!
